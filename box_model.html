      • The CSS Box Model

      The CSS box model is a fundamental concept in web design. It defines how elements are rendered on a webpage as rectangular boxes. Understanding the box model is crucial for controlling the layout and spacing of elements.

      Key Components

      1. Content
      2. Padding
      3. Border
      4. Margin

      CSS Properties

      These properties control the size and spacing of the box model components.

      Property Description Links
      width Sets the width of the content area.
      		Width
      height Sets the height of the content area.
      		 Height
      padding Sets the padding area between the content and the border.
      		 Padding
      border Sets the border around the padding area.
      		 Border
      margin Sets the margin area outside the border.
      		 Margin
